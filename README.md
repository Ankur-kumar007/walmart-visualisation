# walmart-visualisation
This project presents an interactive Power BI dashboard analyzing Walmart sales data. It includes insights on sales performance, profit trends, regional analysis, and category-wise distribution using dynamic visualizations and filters.

This dataset contains historical sales data from Walmart, capturing order-level details across multiple states, categories, and years. It is designed to support sales performance analysis, profitability tracking, and customer purchasing behavior insights.

The dataset includes information such as Order ID, Product Name, Category, Sub-Category, State, Year, Sales, Quantity, and Profit. It enables analysis of regional sales trends, category-wise performance, product-level contributions, and profitability patterns over time.

This data is well-suited for building interactive dashboards in Power BI, using KPIs, slicers, and visual analytics to support data-driven decision making.
